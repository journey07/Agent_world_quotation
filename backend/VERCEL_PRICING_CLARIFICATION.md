# Vercel 플랜 정확한 정보

## ⚠️ 중요: Static Sites vs Serverless Functions

**Static Sites는 완전 무료**이지만, **Serverless Functions는 별도 제한**이 있습니다!

이미지에서 보이는 "$0 USD per month"는 **Static Sites**에 대한 것이며, Serverless Functions를 사용하면 추가 제한이 적용됩니다.

## Hobby 플랜 (무료) - 정확한 제한사항

### Serverless Functions 제한

| 항목 | 제한 |
|------|------|
| **기본 타임아웃** | 10초 |
| **최대 타임아웃 (설정 가능)** | **60초** ⚠️ |
| **메모리** | 최대 2GB |
| **함수 호출** | 100만 회/월 포함 |

### 할당량 초과 시

- **추가 요금 없음** (무료 플랜이므로)
- 하지만 **서비스가 일시 중지**될 수 있음
- 다음 달까지 대기하거나 Pro 플랜으로 업그레이드 필요

## 3D 이미지 생성 기능 분석

### 문제점

- **실제 소요 시간**: 30초 ~ 2분 이상
- **Hobby 플랜 최대**: 60초
- **결론**: Hobby 플랜에서는 **부족할 가능성이 높음**

### 해결 방법 비교

| 방법 | 비용 | 타임아웃 | 추천도 |
|------|------|---------|--------|
| **Vercel Pro 플랜** | $20/월 | 300초 (최대 800초) | ⭐⭐⭐⭐⭐ |
| **Render로 3D만 분리** | 무료 | 무제한 | ⭐⭐⭐⭐ |
| **Hobby 플랜 사용** | 무료 | 60초 | ⭐⭐ (위험) |

## 현재 설정의 문제

`vercel.json`에서 3D 생성 함수를 300초로 설정했지만:

```json
{
  "functions": {
    "api/quote/generate-3d-installation.js": {
      "memory": 2048,
      "maxDuration": 300  // ⚠️ Hobby 플랜에서는 60초로 제한됨!
    }
  }
}
```

**Hobby 플랜에서는:**
- 설정한 300초가 무시되고 **60초로 제한**됨
- 60초 내에 완료되지 않으면 타임아웃 발생

**Pro 플랜에서는:**
- 300초 설정이 정상 작동
- Fluid Compute 활성화 시 800초까지 가능

## 권장 사항

### 무료로 사용하려면

1. **Render로 3D만 분리** (가장 실용적)
   - 3D 생성 엔드포인트만 Render에 배포
   - 나머지는 Vercel Hobby 플랜 사용
   - 타임아웃 걱정 없음

2. **Hobby 플랜 사용 (위험)**
   - 60초 내에 완료되는 경우만 작동
   - 타임아웃 발생 가능성 높음
   - 프로덕션 환경에는 비추천

### 비용을 지불할 수 있다면

**Vercel Pro 플랜** ($20/월)
- 모든 기능 Vercel에서 사용 가능
- 300초 (최대 800초) 타임아웃
- 가장 간단하고 안정적
